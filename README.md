## Unpacking
This supports the following terms to be unpacked

- SMALL_INTEGER_EXT
- INTEGER_EXT
- ATOM_EXT
- MAP_EXT
- NIL_EXT
- STRING_EXT
- LIST_EXT
- BINARY_EXT
- SMALL_ATOM_EXT
- MAP_EXT
